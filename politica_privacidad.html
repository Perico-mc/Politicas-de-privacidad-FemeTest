Política de Privacidad

FemeTest – Tu test de embarazo

Esta aplicación respeta y protege la privacidad de sus usuarias. Esta política describe cómo se recopila, utiliza y protege la información.

1. Información que recopilamos

La aplicación puede recopilar información de forma automática, incluyendo:

  • Identificador de publicidad (ID de anuncios de Google).
  • Datos básicos del dispositivo (modelo, versión del sistema).
  • Preferencias guardadas localmente en el dispositivo.

No recopilamos información personal identificable como nombre, correo electrónico o dirección.

2. Información de salud

FemeTest ofrece contenido informativo relacionado con salud reproductiva femenina. La información ingresada por la usuaria:

  • Se utiliza únicamente con fines informativos.
  • Se almacena localmente en el dispositivo.
  • No se comparte con terceros.

La aplicación no reemplaza el diagnóstico médico profesional.

3. Publicidad

La aplicación utiliza Google AdMob para mostrar anuncios.

AdMob puede recopilar datos para mostrar anuncios personalizados o no personalizados, de acuerdo con las políticas de Google.

Consulta la política de privacidad de Google:

https://policies.google.com/privacy

4. Uso de permisos

La aplicación puede solicitar permisos como:

  • Notificaciones: para recordatorios y avisos programados.

Estos permisos se usan exclusivamente para el funcionamiento de la app.

5. Compartición de datos

No compartimos datos personales con terceros, excepto los necesarios para el funcionamiento de servicios de Google (como AdMob).

6. Seguridad

Tomamos medidas razonables para proteger la información almacenada localmente en el dispositivo.

7. Cambios en esta política

Esta política puede actualizarse. Los cambios se publicarán en esta misma página.

8. Contacto

Si tienes preguntas sobre esta política, puedes contactarnos a través de Google Play.
